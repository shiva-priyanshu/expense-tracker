# 💸 Expense Tracker

A responsive Expense Tracker web application built using HTML, CSS, JavaScript, and Chart.js. It helps users manage their income and expenses, view financial summaries, and analyze spending patterns through interactive charts.

## 🚀 Features

- Add Income and Expense Transactions
- Real-time Balance Calculation
- Category-wise Expense Tracking
- Interactive Doughnut Chart (Chart.js)
- Filter Transactions by Type and Date
- Local Storage Support (Data persists in browser)
- Export Transactions as JSON
- Fully Responsive Design
- Modern UI with Analytics Dashboard

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- Local Storage API


## 📚 Concepts Implemented

This project demonstrates the use of:

- DOM Manipulation
- Event Handling
- Form Validation
- Arrays and Objects
- Local Storage
- Data Filtering
- Dynamic UI Rendering
- Responsive Web Design
- Chart Visualization
